# Aggiungi Cliente

1. Accedi alla sezione "Nuovo Cliente"
2. Compila i campi:
   - Nome cliente (obbligatorio)
   - Email (opzionale)
   - Numero WhatsApp (opzionale)
   - Username Telegram (opzionale, formato @username)
3. Clicca "Aggiungi Cliente"
